# 计算器应用程序

这是一个用Python和Tkinter开发的简单计算器应用程序。

## 功能

- 基本数学运算：加法、减法、乘法、除法
- 清除功能
- 用户友好的界面

## 文件说明

- `计算器.py`：计算器应用程序的第一个版本
- `计算器2.py`：计算器应用程序的改进版本

## 如何使用

1. 克隆这个仓库
2. 运行 `计算器2.py` 文件
3. 使用计算器界面进行计算
